Minor scribbles on Cronbach's alpha

2020-09-04

Some minor scribbles on Cronbach’s alpha:

  1. Cronbach’s alpha is a poor measure of homogeneity (i.e., unidimensionality). It can easily be demonstrated and unidimensional and multidimensional data resolve to the same value of alpha.

  2. Cronbach’s alpha is a poor measure of interrelatedness among items (sometimes labeled “internal consistency”, though this vague term has never been properly defined). This is because alpha is affected by other properties than interrelatedness, viz. test length. Indeed, a common tactic to increase “internal consistency” is to add more items to the test.

  3. Cronbach’s alpha is a poor estimate of reliability (defined in the classical sense as the correlation between parallell forms of a test). What it provides is an estimate of lower bound reliability. Other coefficients do a better job of estimating reliability.

  4. Last but not least, Cronbach’s alpha is not appropriately named. It is a generalization of the Kuder-Richardson Formula 20, and this generalization was described in publications by other authors before Cronbach. Cronbach himself has distanced himself from the name “Cronbach’s alpha”.
