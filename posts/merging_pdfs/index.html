Merging PDF files on MacOS

2020-02-22

I discovered a neat trick for merging PDF files in MacOS. There is a somewhat obscure native Python-script that can accomplish this using the terminal. The script can be accessed via the following command:

"/System/Library/Automator/Combine\ PDF\ Pages.action/Contents/Resources/join.py" -o /PATH/TO/MERGED.PDF /PATH/TO/FIRST.PDF /PATH/TO/SECOND.PDF

This takes the two PDF files FIRST.pdf and SECOND.pdf and combines them into a new file called MERGED.pdf. There isn’t much else to it. Of course, the path to join.py is difficult to remember, but one can solve that problem by creating an alias in ~\.bash_profile. For instance:

alias mergepdf="/System/Library/Automator/Combine\ PDF\ Pages.action/Contents/Resources/join.py --output merged.PDF *.PDF"

With this, one can simply cd into the directory containing the PDF files that need to be merged, and then run the command mergepdf in the terminal. This merges the PDF files in the current directory into the new file merged.pdf. Naturally, the operation is non-destructive, so the original files will remain intact.
