Converting from .djvu to .pdf on macOS

2020-09-27

Once in a while, I’ll encounter a document in djvu-format. Apparently this format was developed as an alternative to the pdf-format in the 1990’s. I’m sure the format has its virtues, but most of the time, what I need is a pdf-file. Fortunately, there is an easy way to convert from djvuto pdf using the application djvu2pdf, which can be installed via Homebrew. Then simply run this command:

djvu2pdf FILE.djvu

where FILE.djvu of course is the location of the djvu-file. This creates a pdf-version of the document. The conversion process is non-destructive.
