PsychRandom.py: A program for randomizing participants to treatment groups

Posted at — Feb 9, 2020

In order to brush up on my Python skills, I decided to write a small utility program that randomizes participants to treatment groups for psychology experiments. Not really groundbreaking work, but hey, it might prove useful at some point.

The program is called PsychRandom.py, and can be run either from a GUI or from the command line. The user specifies the number of participants in the experiment, as well as the number of treatment groups and whether or not a balanced design is required (i.e., equal number of participants in each treatment group). The program then randomly assigns a treatment group to each participant and outputs a csv-file that looks similar to this:

subject_id treatment_group
1 2
2 1
3 2
4 1
5 1

Of course, this program would primarily be useful in scenarios where the experiment is not conducted in front of a computer, as the usual programs for running psychology experiments (e.g., Matlab) tend to have built-in support for randomization.

PsychRandom can be downloaded via GitHub: https://github.com/amospagin/psychRandom. I suppose the program requires some previous familiarity with Python in its current form.
