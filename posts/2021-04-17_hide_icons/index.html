Hiding desktop icons on macOS

2021-04-17

Although I’ll put files on the macOS desktop, I usually don’t want to actually see the files. They just look like clutter, and I don’t access them by pointing and clicking anyway. Here’s a terminal command for hiding the desktop icons:

defaults write com.apple.finder CreateDesktop false

Enter this, and then run killall Finder. To enable the icons, just run the same command, but change false to true.
