Advantages of programmatic statistical computing

2020-02-17

I was recently reminded of why taking the time to learn a programming language for statistical computing (e.g., R) is such a valuable investment. I had completed the data collection phase for a study, and realized that I wanted to plot the data for each participant individually. Because each participant (n = 80) had completed three separate experimental phases, this meant that I would need to produce 80 * 3 = 240 separate plots.

That’s a lot of plotting.

Thankfully, this was pretty easy to accomplish/automate in R. I created an RMarkdown document, and wrote a function specifying the parameters of the plots (e.g., label titles, axes, and so on). Then I wrote a for-loop that applied the function to all instances in the data file, and processed the output using knitr and pandoc. The result was a 24-page document with neatly ordered plots.
